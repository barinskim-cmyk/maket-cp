Fix Приоритет Проблема Экран Описание Решение Заметки Masha
Критичные (аудит preview loading)
Высокий Race condition: параллельные sync Cloud sync sbUploadPreviews() и sbUploadProject() могут запускаться одновременно. Риск: частичные загрузки, повреждение данных в облаке. Добавить sync queue
Высокий Тихий провал загрузки превью Cloud sync Если загрузка превью падает, _pendingSync не устанавливается. Пользователь думает что данные в облаке. Добавить _previewsPendingSync + retry
Высокий async/await в ES5 коде shootings.js Строки 140, 199 используют async/await. Ломает ES5-браузеры (IE11, старый Safari). Должно быть .then()/.catch(). Заменить на .then()
Средние
Средний FileReader ошибки не ловятся Превью previews.js:603-639 -- ошибки чтения файлов пропускаются без уведомления. Добавить onerror обработчик
Средний IndexedDB quota exceeded Превью Если IndexedDB переполнена -- ошибка пропускается без уведомления пользователя. Показать предупреждение
Средний Дедупликация непоследовательна Превью Разная логика дедупликации в desktop и browser режимах. Унифицировать
UX / polish
Низкий Нет удаления проекта из списка Съёмки Нельзя убрать проект из списка, только закрыть. Накапливаются. Кнопка удаления
Низкий Canvas memory leak Превью Canvas не освобождается при перезагрузке галереи. При 1000+ фото может замедлить. Cleanup при rerander
