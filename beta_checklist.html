Fix Приоритет Проблема Экран Описание Решение Заметки Masha
UX: проекты и навигация (из комментариев Masha)
Высокий Формат имён съёмок Съёмки Названия съёмок должны быть в формате Бренд_YYYY_MM_DD. Сейчас свободный текст. Автоформат из полей бренд + дата
Высокий Soft delete проектов Съёмки Нет кнопки удаления проекта. Проекты накапливаются. Нужен soft delete (в базе остаётся, из списка пропадает). Кнопка удаления + поле deleted_at в БД + фильтр "Скрытые"
Высокий Убрать ручную загрузку/выгрузку из облака Съёмки Кнопки "Из облака" не нужны -- всё должно синхронизироваться автоматически. Скрыть завершённые проекты. Убрать кнопки, добавить фильтр "Скрыть завершённые"
Высокий Командный доступ к проектам Съёмки Нет обмена съёмками между участниками команды (стилист, фотограф). Один создаёт проект -- у другого он не отображается. Роль "team member" + проект виден всем участникам
Средний Почистить этапы пайплайна Съёмки Сейчас 8 этапов, но часть пока не покрываем. Оставить: от преотбора до "передано на ретушь". Скрыть/деактивировать неготовые этапы
UX: карточки и превью (из комментариев Masha)
Высокий Шаблоны: нет меню выбора и управления Контент Шаблоны сохраняются непонятно куда. Нет окна просмотра шаблонов, нет меню выбора из сохранённых. Нужно сохранить реальные шаблоны из проектов. Окно управления шаблонами + меню выбора
Высокий Drop zone: drag в любую область экрана Превью Когда нет фотографий, перетаскивание должно работать на весь экран. Сейчас маленький контейнер -- неудобно. Fullscreen drop overlay при пустой галерее
Средний Режим карусели в desktop (как в мобилке) Контент В мобилке есть пролистывание внутри карточки -- круто. Хочется так же в desktop как альтернативный режим. Переключатель grid/carousel в desktop
Средний Редактирование имени карточки Контент Нет возможности вручную редактировать имя карточки. Должно поддерживать автопереименование из чек-листа артикулов. Редактируемое поле имени + авто из артикулов
Средний Глюк layout в раскладке карточек Контент Layout engine работает в редакторе, но глючит в общем раскладе карточек (список всех карточек). Отладить рендер карточек в списке
Средний Фильтр по рейтингу в карточках Контент Фильтрация по рейтингу есть в превью, но не в карточках. Должна быть доступна в desktop. Добавить pv-filter в карточки
Низкий PDF экспорт на все экраны Контент PDF экспорт работает для карточек. Хочется экспорт всего селекта (не только карточки). Расширить PDF на allcontent
Низкий Убрать "Экспорт списка" и названия в сайдбаре Контент Кнопка "Экспорт списка" в сайдбаре лишняя. Названия карточек в списке -- тоже лишнее. Убрать кнопку + упростить сайдбар
UX: доступ и роли (из комментариев Masha)
Высокий Роли в share-ссылках: права и отображение Share Непонятно где отображаются метки ссылок. Изменения должны фиксироваться по роли. Согласовать может только заказчик. Нет разграничения прав. UI меток + role-based permissions (только client approve)
Технические (аудит preview loading)
Высокий Race condition: параллельные sync Cloud sync sbUploadPreviews() и sbUploadProject() могут запускаться одновременно. Риск: частичные загрузки, повреждение данных. Добавить sync queue
Высокий Тихий провал загрузки превью Cloud sync Если загрузка падает, _pendingSync не устанавливается. Пользователь думает что данные в облаке. Добавить _previewsPendingSync + retry
Средний async/await в ES5 коде shootings.js Строки 140, 199 используют async/await. Может сломать старые браузеры. Заменить на .then()
Низкий FileReader ошибки не ловятся Превью Ошибки чтения файлов пропускаются без уведомления. Добавить onerror обработчик
Низкий Canvas memory leak Превью Canvas не освобождается при перезагрузке галереи. При 1000+ фото может замедлить. Cleanup при rerender
