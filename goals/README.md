# /goals/

Координаты CoS-артефактов для execution loop (`strategy-2026.md`, `agent-team-v2.md` фаза 7).

## Содержимое

- `current.md` — текущая Goal card актуального цикла. Формулируется Машей, записывается CoS, подписывается Машей.
- `archive/cycle-N.md` — архив завершённых goal card'ов по итогам monthly review (появляется после закрытия цикла, не заранее).

## Правила

1. Один активный `current.md` одновременно. Новый цикл перезаписывает текущий после того, как предыдущий отправлен в `archive/`.
2. Goal card — ≤ 1 страница, 3-4 предложения от Маши + CoS structure (Ставка → Цель → Success criteria → Отложено → Approval).
3. Подпись Маши — обязательна до старта task proposals. Без подписи — цикл не запускается.
4. Измеримый результат важнее красивой формулировки. "Первый чек [anchor client] получен" лучше, чем "укрепить anchor-позицию".

## Связанные артефакты

- `/tasks/cycle-N-review.md` — task proposals под текущую goal card.
- `/project-log.md` — еженедельный лог с отсылкой на действующий goal.
- `/reviews/monthly-YYYY-MM.md` — monthly review.
