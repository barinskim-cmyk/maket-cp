1. Supabase -- создание проекта
supabase.com -- бесплатный план хватит для старта
Можно через GitHub или email
Название: maketcp, Регион: Frankfurt (eu-central-1) -- ближе к РФ, Пароль БД: любой надежный
Settings > API > Project URL
Settings > API > Project API Keys > anon / public
Видно в URL или в Settings > General
2. Запустить SQL-схему базы данных
Создает все таблицы, политики безопасности и функции
Левое меню > SQL Editor > New query
Файл лежит в проекте. Скопировать весь текст, вставить в редактор, запустить.
Table Editor (левое меню) -- должны быть: profiles, projects, cards, slots, comments, share_links, project_members, stage_events
3. Supabase Storage -- бакеты для файлов
Хранение миниатюр и оригиналов фото
Storage > New bucket > Name: thumbnails, Public bucket: включить
Storage > New bucket > Name: originals, Public bucket: выключить
4. Аутентификация
Magic link -- клиенты заходят по ссылке из email без пароля
Authentication > Providers > Email > Enable. Убедись что "Confirm email" включен.
В том же разделе Email provider: Enable Magic Link = ON
Authentication > URL Configuration > Redirect URLs. Добавить: http://localhost:8080, http://127.0.0.1:8080
