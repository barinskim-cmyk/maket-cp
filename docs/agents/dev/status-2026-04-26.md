**Что сделано:**

- Кнопки 168px min-width «Подтвердить» (primary) / «Не тот артикул» (outlined)
- `box-sizing: border-box` + `object-fit: contain` на thumbs — фото сохраняют пропорции внутри fixed-box

**Existing flow сохранён:**
- `arToggleVerify(idx)` мутирует `proj.articles[i].status`
- `_arApplyCardRename` + `sbAddMatchDecision` cloud sync
